BuckleScript bindings for [JsBarcode](https://github.com/lindell/JsBarcode).

# Installation
```
  npm install bs-canvas bs-barcode
```

# :hammer: Build

```
npm run build
```

# :eyeglasses: Build + Watch

```
npm run start
```

# :pencil2: Editor

If you use `vscode`, Press `Windows + Shift + B` it will build automatically
